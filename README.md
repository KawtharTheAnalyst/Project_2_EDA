# Project 2: Exploratory Data Analysis (EDA) - Lagos Healthcare Dataset.

### Project Overview
This project performs an exploratory data analysis on the Lagos Healthcare dataset (9,305 patient records). The goal was to uncover pattern, identify outliers and provide actionable recommendations to improve clinical monitoring protocols.

### Tools Used
* **Python** (Pandas for data manipulation, Matplotlib/Seaborn for visualization).

### Key Insights
* **Distributions:** Most health metrics (BMI, Systolic/Diastolic BP, Cholesterol) are right-skewed requiring median-based analysis.
* **Outliers:** Significant outliers were identified in BMI and vital signs suggesting the need for data quality audits or targetted patient care.
*  **Clinical Trends:** Patients diagnosed with **Artery Disease** and **Heart Failure** showed notably higher average cholesterol levels compared to other diagnosis.

### Files Included
* **Project_2_EDA.ipynb**: Full analysis process, data visualization and correlation matrix.
* **Project_2_EDA.pdf**: PDF export of the analysis notebook.
* **Project_2_EDA_Report.pdf**: Detailed summary of findings and business recommendations.

### Recommendations
* Implement continued cholesterol monitoring as part of standard cardiac care protocols based on identified high-risk trends.
